## 🏃‍♀️실행방법
---
1. git clone 
2. clone한 폴더로 이동
3. cmd에서 `pip install -r requirements.txt`로 필요 라이브러리 설치 후 환경 설정
4. `python main.py` 실행 후 타이핑 하고 싶은 부분에 커서 놓기
5. 화면을 보면서 수어 지문자를 표시하면서 타이핑 진행
